# LinkedIn Post — Data Analytics Internship Completion

---

Excited to share that I have successfully completed my 5-week Data Analytics Internship at ApexPlanet Software Pvt. Ltd.!

Here is what I built across 5 weeks:

**Week 1 — Data Immersion & Wrangling**
Cleaned and prepared a 9,994-row Superstore dataset. Handled missing values, fixed data types, removed duplicates and engineered new features (Profit Margin, Order-to-Ship Days).

**Week 2 — Exploratory Data Analysis & Business Intelligence**
Performed deep EDA with statistical distributions, built an SQLite database with SQL queries, and created an interactive HTML BI dashboard with segment and regional insights.

**Week 3 — Advanced Analytics & Interactive Dashboard**
Built RFM customer segmentation, cohort retention analysis, time-series decomposition, and a multi-page interactive dashboard with live JavaScript filters and animated charts.

**Week 4 — Data Storytelling & Statistical Validation**
Crafted 5 data narratives validated by hypothesis testing — Shapiro-Wilk, T-Test (segments), ANOVA (regions), Chi-Square (shipping) and Pearson Correlations — with a premium story presentation.

**Week 5 — Capstone Portfolio Finalization**
Consolidated everything into a professional GitHub portfolio with a GitHub Pages website linking all 4 repositories.

Key insight from this journey: The "Discount Trap" — products discounted above 40% almost always generate losses. A finding that went from raw CSV to statistically validated business recommendation.

Tools used: Python, Pandas, NumPy, SciPy, Matplotlib, Seaborn, SQLite, HTML/CSS/JavaScript, GitHub.
